## Why

Психологические курсы по гештальт-терапии нуждаются в отдельной рекламной точке входа: простой одностраничный сайт, который объясняет ценность курсов, ведёт к заявке и позволяет измерять эффективность рекламы. Сейчас такого публичного лендинга нет — без него сложно запускать трафик и понимать, какие блоки и каналы конвертируют.

## What Changes

- Создать одностраничный маркетинговый сайт (лендинг) на русском языке с акцентом на гештальт-терапию
- Добавить ключевые секции: герой, о подходе, программа курсов, преимущества, отзывы/доверие, FAQ, форма заявки или CTA
- Подключить веб-аналитику (Яндекс.Метрика) для отслеживания визитов, целей и кликов по CTA
- Настроить публикацию на GitHub Pages из репозитория проекта
- Обеспечить адаптивную вёрстку и быструю загрузку на мобильных устройствах
- Подготовить базовую SEO-разметку (title, description, Open Graph) для шаринга в соцсетях

## Capabilities

### New Capabilities

- `landing-page`: Одностраничный сайт с контентом о психологических курсах по гештальт-терапии, навигацией по секциям и призывами к действию
- `analytics-integration`: Подключение Яндекс.Метрики, инициализация счётчика и отслеживание ключевых целей (клики CTA, отправка формы, скролл до секций)
- `github-pages-deployment`: Сборка и публикация статического сайта на GitHub Pages с корректными путями и автоматическим деплоем при push

## Impact

- Новый репозиторий/проект: статические HTML/CSS/JS (или лёгкий генератор) в корне `PsyConstructor`
- Внешние сервисы: Яндекс.Метрика (нужен ID счётчика), GitHub Pages
- CI/CD: GitHub Actions workflow для деплоя
- Контент: тексты, изображения и контакты для лендинга (могут быть заглушками на первом этапе)
- Нет изменений существующих API или бэкенда — только статический фронтенд
