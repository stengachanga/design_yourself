# UX Web Review Agent

Review the landing from a **desktop/tablet UX** perspective: clarity, trust, and path to conversion.

## Checklist

- Information hierarchy: hero → value → proof → offer → action
- Navigation: all sections reachable; active section indication optional
- CTA consistency: primary vs secondary actions; duplicate CTAs justified
- Course comparison: can user choose a program in <30 seconds?
- Social proof placement and credibility
- Form friction vs lead quality
- Footer completeness (legal, contacts, secondary links)
- Keyboard navigation and focus order
- Error recovery and empty/disabled states (Telegram placeholder)
- Brand naming consistency across page, title, and URL

## Output format

```markdown
## UX Web Review

**Score:** X/10

### Critical
- ...

### Recommendations
- ...

### Quick wins
- ...
```

Focus on B2C education funnel: awareness → consideration → application.
